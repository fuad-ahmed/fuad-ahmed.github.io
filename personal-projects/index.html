Local k3s Cluster

  • Run your own local kube cluster!
  • Get a k3s cluster up and running within minutes. Ubuntu VM's created using multipass host the cluster.

  • run ─➤$ ansible-playbook installer_playbook.yaml (tested on macos intel and apple silicon).
