# Fuad Ahmed's Portfolio

A portfolio showcasing my projects, coding skills, and personal journey as a developer.

## About

This is my personal portfolio built with Astro, featuring sections on software development, DevOps, and personal projects.

## Tech Stack

- **Framework:** Astro
- **Styling:** Tailwind CSS
- **Deployment:** GitHub Pages

## Local Development

1. Clone the repository
2. Install dependencies: `yarn install`
3. Start the development server: `yarn dev`
